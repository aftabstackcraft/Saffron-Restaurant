# Saffron & Sage Restaurant Website

A polished static restaurant website for Saffron & Sage, a modern Indian kitchen. The site is built with separate HTML, CSS, and JavaScript files, and includes five complete pages with responsive design, restaurant photos, menu filtering, and a reservation contact form.

## Pages

- Home - Restaurant introduction, highlights, location details, opening hours, and featured photos.
- Menu - Signature dish section plus filterable menu categories for starters, mains, desserts, and drinks.
- Services - Dine-in, private dining, catering, chef's table, party menus, and takeaway services.
- About - Restaurant story, values, and timeline.
- Contact - Address, contact details, reservation form, and location-style map panel.

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Static image assets

## Features

- Fully responsive layout for desktop, tablet, and mobile screens
- Separate files for HTML, CSS, and JavaScript
- Fixed navigation header with mobile menu toggle
- Filterable menu section using JavaScript
- Contact form validation with success/error messages
- Restaurant-style photo assets
- Clean modern Indian restaurant design

## How To Run

Open `index.html` directly in your browser:

```text
outputs/index.html
```

You can also serve the folder with any static server and open:

```text
http://localhost:PORT/index.html
```

## Customization

You can update restaurant details in the HTML files:

- Restaurant name
- Address
- Phone number
- Email
- Menu item names and prices
- Opening hours
- Service descriptions

Design changes can be made in:

```text
css/styles.css
```

JavaScript behavior can be updated in:

```text
js/script.js
```
